Set up a Mock Election: The button will create a simple election with predetermined properties to test the system, which include the following:

Starting time: The election will start immediately after its creation.
Ending time: The election will end after a fixed time (default: three days).
Verification code: The verification code will be either provided entirely by the system, or the user will provide part of it (default: user provides part of the code).
Miscellaneous: The election title, its description as well as the question and possible choices are already set.

Create Election: This will show an advanced settings page where a customized election can be created.

Close Election: The selected election will be closed after confirming the set password (if one was set).

Remove Election: Removes the selected election from the list. If the voting results should be saved, the election has to be closed first.

Invite Voters to Vote: After selecting a running election the button will be displayed, providing a link to the voting procedure.

Check Election Result: After selecting a closed election the button will be displayed, providing a link to view the results.
